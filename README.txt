
# CONTROL QAQC - Densidad In Situ

Este sistema permite ingresar y registrar ensayos de densidad in situ.

## Pasos para usarlo en Streamlit Cloud:
1. Sube esta carpeta a un repositorio de GitHub.
2. Inicia sesión en https://streamlit.io con tu cuenta de Gmail.
3. Crea una nueva app conectando tu repositorio.
4. Define como archivo principal: `app.py`
5. ¡Listo! Tu sistema estará en línea.
